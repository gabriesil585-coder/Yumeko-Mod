# Mod Yumeko - Sistema de Cassino para Minecraft Bedrock

## Descrição
Mod Yumeko é um add-on para Minecraft Bedrock que adiciona um sistema completo de cassino e apostas inspirado no anime Kakegurui. O mod inclui blocos de jogos, entidades especiais, itens exclusivos e mecânicas de jogo únicas.

## Características
- **Blocos de Jogo:**
  - Mesa de Apostas (Gambling Table)
  - Máquina Caça-Níqueis (Slot Machine)
  - Trocador de Fichas (Chip Exchange)

- **Entidades:**
  - Yumeko (Gerenciadora principal)
  - Mary (Assistente)
  - Kirari (Administradora)
  - Manager (Gerenciador)

- **Itens:**
  - Fichas de Poker (Azul, Vermelha, Dourada)
  - Token VIP
  - Baralho de Cartas

## Instalação
1. Extraia os arquivos
2. Copie `Yumeko_BP` para sua pasta de Behavior Packs
3. Copie `Yumeko_RP` para sua pasta de Resource Packs
4. Ative os packs em seu mundo Minecraft

- Versão: 1.0.0
- Compatibilidade: Minecraft Bedrock 1.20+
